# Calculator

A command-line calculator built with Node.js.

## Features

- Addition, Subtraction, Multiplication
- Natural Log (ln) and Log Base 10
- Percentage calculations
- Trigonometric functions (Sin, Cos, Tan) — input in degrees

## Usage

```bash
node calculator.js
```

Select an operation from the menu, enter your numbers, and see the result. The calculator loops until you choose Exit.

## Requirements

- Node.js
